# Cava IAF

CRM- og logbogssystem med AI-assistent til rådgivningsvirksomheder.

## Struktur

- **frontend/** — React + Vite SPA med Supabase-autentificering
- **backend/** — Express API-server med Supabase, Anthropic og Dinero-integrationer

## Kom i gang

### Frontend

```bash
cd frontend
cp .env.example .env   # udfyld miljøvariabler
npm install
npm run dev
```

### Backend

```bash
cd backend
cp .env.example .env   # udfyld miljøvariabler
npm install
npm run dev
```
